# 💧 Smart Water Usage Analytics System

# 📖 Overview

The **Smart Water Usage Analytics System** is a Python-based data analytics project that simulates and analyzes water consumption across multiple hostels. It generates realistic daily water usage records, tank levels, and water quality parameters to support sustainable water management and operational planning.

The system creates a comprehensive dataset spanning multiple years, making it suitable for analytics, visualization, forecasting, and machine learning applications.

# ✨ Features

- 💧 Daily Water Usage Simulation
- 🏢 Multi-Hostel Monitoring
- 🚰 Tank Capacity & Tank Level Tracking
- 🧪 Water Quality Simulation
- 📅 Multi-Year Dataset Generation

# 📊 Dataset Features

Each record contains:

- Date
- Hostel Name
- Water Usage (Liters)
- Tank Capacity
- Tank Level
- Total Rooms
- Total Students
- pH
- Turbidity
- TDS
- Hardness
- Iron
- Chlorides
- Residual Chlorine
- Fluoride
- Sulfate
- Nitrates
- Chloramines
- Alkalinity
- Coliform Bacteria

---

# ⚙️ How It Works

1. Initialize hostel information.
2. Generate daily water usage.
3. Simulate tank levels.
4. Generate water quality values.
5. Store all records in a Pandas DataFrame.
6. Export the final dataset as CSV.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ATUL-K-VERMA/Smart-Water-Usage-Analytics.git
```

Navigate to the project

```bash
cd Smart-Water-Usage-Analytics
```

Install dependencies

```bash
pip install pandas numpy
```

Run the project

```bash
python main.py
```

The generated dataset will be saved as:

```
water_usage_dataset.csv
```
